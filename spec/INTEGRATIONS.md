# Integrations

## Jira
- Ingest ticket by URL
- Read fields: summary, description, acceptance criteria, attachments
- Update status as workflow progresses
- Write back links to PRs, QA reports, release notes

## Web Intake
- Fetch requirement source from URL
- Snapshot content and extract requirement summary

## GitHub
- Create branches and PRs
- Retrieve CI checks and test status
- Enforce required reviews before merge
- Post review comments and approvals

## Coralogix
- Send workflow events (state changes, approvals)
- Send test metrics and deployment markers
- Alert on gate failures or repeated loopbacks

## Secrets
- Store tokens in a secrets manager
- Rotate on schedule; log access events
